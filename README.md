### Install sqlc

### MacOs

`brew install sqlc`

### Ubuntu

`sudo snap install sqlc`

### Go Install

#### Go >= 1.17:

`go install github.com/sqlc-dev/sqlc/cmd/sqlc@latest`

#### Go < 1.17:

`go install github.com/sqlc-dev/sqlc/cmd/sqlc@latest`
